• Key Terms From Chapter 2

  • External Style Sheet

    An external style sheet saved as a separate text document but linked to from the webpage it modifies.

  • ID Selectors

    A specific type of Cascading Style Sheets selector intended to be used once on a webpage. It begins with the # symbol.

  • Inline Style

    A formatting declaration created in the HTML tag being formatted.

  • List Item

    One item inside an ordered or unordered HTML list.

  • Nested

    HTML tags that are created between other HTML tags.

  • Ordered List

    HTML tag set used to create a numbered or lettered list of similar items, in order.

  • Relative Links

    Links that do not contain the full file path to a webpage.

  • Unordered Lists

    HTML tag set used to create a bulleted list of items.

  • Version Control

    Software that is used to create date-time stamped copies of coding projects.

  • WYSIWYG (What You See Is What You Get)

    An editor that shows the user "What You See Is What You Get."
